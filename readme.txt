Empower your Website with myCred Github! Reward users with points for approved pull requests on designated repositories. Easy GitHub integration.

== Description ==

[myCred Github](https://mycred.me/store/mycred-github-rewards/) is a plugin for WordPress that enables you to gain points based on approval of his/her pull request with the master branch of the project.

= Features =
* It allows admin to get points once his/her pull request is accepted
* Ease of connection between your site and the github by creating a github app and get you
through github token, username, client ID and client secret.
* Allow admin to view all repositories in his website and define which repositories that gain points
will apply on it.
